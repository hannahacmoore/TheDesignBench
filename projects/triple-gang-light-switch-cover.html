Triple Gang Light Switch Cover

A triple-gang switch cover plate covering every practical combination of decora/rocker cutouts and blank panels, useful for setups where not all three gangs are wired to a switch. Each combination is available in a standard size and a slightly oversized "Large" version (+0.5cm on all sides, useful for paint buildup or slightly oversized wall openings). Most combinations also come in Version 1 (standard screw mounts) and Version 2 (exposed screw mounts removed on flat sections for a cleaner look), Triple Decora is the only style without a Version 2, since it has no flat section to clean up. The "end switch only" and "end blank only" files are designed symmetrically, flip the plate 180° after printing to get the mirrored version (e.g. the "left switch only" file becomes "right switch only" when flipped), so one file covers both mirrored cases. Designed in SolidWorks.

Triple Decora

All three positions have a decora/rocker cutout. No flat section to clean up, so this is the only style without a Version 2. Includes a standard size and a +0.5cm oversized "Large" size.

Triple Blank Version 1

All three positions are blank, for a fully unused or decorative plate. Standard screw mounts. Includes a standard size and a +0.5cm oversized "Large" size.

Triple Blank Version 2

Same fully blank layout as Version 1, with the exposed screw mounts removed for a cleaner look. Includes a standard size and a +0.5cm oversized "Large" size.

Center Switch Only Version 1

A single decora/rocker cutout in the center position, blank on both ends. Standard screw mounts. Includes a standard size and a +0.5cm oversized "Large" size.

Center Switch Only Version 2

Same center-switch layout as Version 1, with the exposed screw mounts on the flat sections removed for a cleaner look. Includes a standard size and a +0.5cm oversized "Large" size.

End Switch Only Version 1

A single decora/rocker cutout at one end, blank across the other two positions. Symmetrical design, flip 180° after printing for the mirrored (opposite end) version. Standard screw mounts. Includes a standard size and a +0.5cm oversized "Large" size.

End Switch Only Version 2

Same end-switch layout as Version 1, with the exposed screw mounts removed for a cleaner look. Flip 180° after printing for the mirrored version. Includes a standard size and a +0.5cm oversized "Large" size.

Center Blank Only Version 1

A single blank position in the center, decora/rocker cutouts on both ends. Standard screw mounts. Includes a standard size and a +0.5cm oversized "Large" size.

Center Blank Only Version 2

Same center-blank layout as Version 1, with the exposed screw mounts on the flat section removed for a cleaner look. Includes a standard size and a +0.5cm oversized "Large" size.

End Blank Only Version 1

A single blank position at one end, decora/rocker cutouts across the other two. Symmetrical design, flip 180° after printing for the mirrored (opposite end) version. Standard screw mounts. Includes a standard size and a +0.5cm oversized "Large" size.

End Blank Only Version 2

Same end-blank layout as Version 1, with the exposed screw mounts removed for a cleaner look. Flip 180° after printing for the mirrored version. Includes a standard size and a +0.5cm oversized "Large" size.
