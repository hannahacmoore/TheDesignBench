Double Gang Light Switch Cover

A double-gang switch cover plate available in three cutout combinations: both decora/rocker cutouts, both blank (no cutout), or one decora + one blank for mixed setups (e.g. one live switch, one unused or decorative slot). Each combination includes a standard size and a slightly oversized "Large" version (+0.5cm on all sides) for covering paint buildup or slightly oversized wall openings. The mixed (decora + blank) version also comes in two file styles: v1 with standard screw mounts, and v2 with the unnecessary exposed screw mounts removed on the flat section for a cleaner look. Designed in SolidWorks.

Double Decora

Both cutouts are decora/rocker-style, for two switches side by side. Includes a standard size and a +0.5cm oversized "Large" size.

Double Blank

Both sides are blank (no cutout), for unused or decorative double-gang plates. Includes a standard size and a +0.5cm oversized "Large" size.

Mixed Version 1

One decora/rocker cutout and one blank side, for mixed setups. Standard screw mounts. Includes a standard size and a +0.5cm oversized "Large" size.

Mixed Version 2

Same mixed layout as v1, with the exposed screw mounts removed on the flat section for a cleaner look. Includes a standard size and a +0.5cm oversized "Large" size.
