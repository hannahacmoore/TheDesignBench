Philips Hue Remote Holders

Every Hue remote holder size in one place, single, double, and triple gang wall plates for a Philips Hue Smart Button/Dimmer Remote, each combined with blank and/or Decora/rocker cutout sections in standard and oversized sizes. The holder shape is directional, use your slicer's Mirror tool (not Rotate) for a mirrored layout. Requires 2x 10mm x 2mm magnets per print (not included). Every style below opens its own "View Design" links.
